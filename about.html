Technologies Used

We have used Bootstrap 4 since it is designed to be responsive to mobile devices. Mobile-first styles are part of the core framework. CSS is used to style the page, however bootstrap is doing the heavy-lifting here. HTML is our structure. Javascript is used to add functionality to the page.

How To Use

Since it is using local storage, it is completely independent and doesn't need a Database to store the data. I would recommend using this project manager when you have some project which require deep work and strict abstaining from the internet.
